# HTTPStatus

A clean, searchable HTTP status code reference. Find any status code and get a plain-English explanation, when to use it, common mistakes, and a real-world example.

## Features

- **30 status codes** across all categories — 1xx, 2xx, 3xx, 4xx, 5xx
- **Search** by code number or name
- **Category color-coding** — green for success, amber for redirects, orange/red for errors
- **Practical details** — not just definitions, but when to use it and what to avoid
- **Two-column layout** — sidebar list + full detail panel
